CodePad Tutorial

Introduction

tan90 is an easy-to-use platform for technical interview and face-to-face tutoring in practical programming. I will walk you through the very basics that you need to utilize this platform.

Interface


Split at a default 7 to 5 ratio left to right, the tan90 CodePad interface consists of an editor, a console and a video chat panel (which by default is blank). When performing solo coding, one can simply type in the editor and click the "run" button at the top right corner of the editor to execute his code and view output of the code displayed at the console.
When connected to a peer through video chat, a video box will pop up on top of the console box so that the two coders can see and talk to each other while coding.

Video Chat


To initiate a video chat with an arbitrary peer who is also using the tan90 CodePad, one only needs to perform the following steps:

  1. Click the inverted triangle at the top right corner, and click again the code in the dropdown. The code in the dropdown will be automatically copied to clipboard
  2. Send the code to a friend who you would like to connect with
  3. Your friend will paste the code into the "PEER CODE" box and click "connect"


That's it! The two of you should now be able to see and talk to each other.

Code Sharing

What's more? You can share your code to a peer so that the two of you can work on the same piece of code synchronously. To share your code, just follow the steps here:

  1. Click the "share" button, you should see a pop up. If you see something like "waiting for code", just close the pop up and try again
  2. Now, the box within the pop up window contains your code that your friend is able to connect to you with. Just click "copy my code" and the code will be automatically copied to your clipboard
  3. Send the code to a friend who you would like to share your code with
  4. Your friend will open up the same pop up window and paste the code into the only input box
  5. Click "SYNC" and the two of you will see each other typing in the same document

Creating Code Room

Now, it is not always convenient to share the code between you and your friend when the two of you are already using the CodePad. It would be wonderful if the two of you could just create a room beforehand that is specifically for the two of you, and when both of you are in the Code Room, a video chat can be initiated without having you send the code to your friend. Tan90 allows you to do exact that!

When you are at this page, here are the steps to create a Code Room:

  1. Enter a name for the room (in the box of course)
  2. Click "Create new room"
  3. Tell your friend the name of the room you created (which shold definitely be easy to remember)
  4. Both you and your friend can visit the page "tan90.tech/room/yourRoomName}" and start coding!

Note that when both of you are using the same Code Room (visiting the same page), you will be notified when your friend enters the room. To keep distractions away from you (also because that we are working on a better notification system), you will know your friend is ready to connect when you see that the box where you are supposed to be inputting your friend's special code is automatically filled up, like this:


Don't be too surprised when you see this magic happening. The code that you see is that of your friend, and you can just click "connect" to meet your friend so long as the code is still there (that your friend is still in the Code Room).
